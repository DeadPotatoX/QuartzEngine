Quartz Engine!

What is Quartz Engine

Quartz is a game engine, programmed in Java, with utilities for rendering, etc (more utilities coming soon). It is a stable platform to build games off of. The reasoning behind the name QuartzEngine is that Quartz timers are a stable source of timing.

How do I use it?

Follow the tutorials on the Wiki and not long after you will have a simple game setup! QuartzEngine is open-source, meaning you can see all the code on GitHub! It is free to use, and you can use it commercially*

Comming Soon

What is planned for QuartzEngine? Tiles, Sprites, SpriteSheets, Basic Players, an Entity Framework, bullet physics and Advanced Gravity! Submit your ideas on the Wiki!

Author
